# P0 上线验收记录

更新时间：2026-06-14

> 后续闭环状态：本文记录的旧 smoke 断言和 `scrollLeft` hint 已在 P2/P3 处理；最终 `npm.cmd run verify`、九语言 smoke 和上线审计均通过。

## 验收范围

本轮只处理 P0 上线阻断项：

- 构建是否失败
- sitemap URL 是否可访问
- indexable 页面是否存在 noindex、canonical、H1、JSON-LD 等阻断错误
- 公开语言首页是否被错误 noindex 或 canonical 到其他语言
- 尾斜杠 URL 是否造成 404 或重复入口
- 可见内部链接是否出现 404
- robots 是否阻断当前页面所需关键资源

## 已修复 P0

### 1. 简体中文首页被错误 noindex

问题：

- `/zh-cn` 是公开语言入口，但返回 `noindex,follow`。
- `/zh-cn` canonical 指向英文首页。
- 英文首页 hreflang 缺少 `zh-Hans`。
- sitemap 不包含简体中文入口。

修复：

- 将 `zh-cn` 纳入 sitemap 语言配置。
- 新增 `/sitemap-zh-cn.xml`。
- 将简体中文首页加入本地化 SEO 分组。
- 补齐 `zh-Hans` hreflang。

结果：

- `/zh-cn` 返回 `index,follow`。
- `/zh-cn` canonical 为 `https://aibeautystylist.com/zh-cn`。
- 首页 hreflang 包含 `zh-Hans`。
- sitemap URL 数量从 242 增加到 243。

### 2. 语言首页尾斜杠 URL 返回 404

问题：

- `/zh-cn/` 等语言首页带尾斜杠时返回 404。
- 项目 canonical 和 sitemap 使用无尾斜杠 URL，因此带尾斜杠 URL 应统一规范化。

修复：

- 将 Astro 路由匹配改为 `trailingSlash: "ignore"`。
- 在 middleware 中对 GET/HEAD 的带尾斜杠页面 URL 统一 308 到无尾斜杠版本。

结果：

- `/zh-cn/` 308 到 `/zh-cn`。
- `/de/` 308 到 `/de`。
- `/looks/soft-glam/` 308 到 `/looks/soft-glam`。

### 3. 可见内部链接存在 404

问题：

- 部分英文 SEO 页 related links 指向不存在页面。
- 多语言 SEO 页 related links 在没有对应本地化 SEO 页时，错误生成了 `/de/ai-makeup-try-on`、`/fr/virtual-makeup-app` 等不存在路径。

修复：

- 英文静态 SEO 页将不存在的 related URL 替换为已有相近页面。
- 多语言 related links 优先使用已存在的本地化 SEO URL。
- 如果目标是可本地化应用路由，则使用本地化应用路由。
- 如果目标没有本地化版本且不是应用路由，则回退到英文 URL，避免 404。

结果：

- sitemap 页面中的可见内部链接检查：401 个链接，0 个 404。
