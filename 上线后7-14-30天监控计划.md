# 上线后 7/14/30 天监控计划

更新时间：2026-06-14

## 上线当天

| 检查 | 通过标准 |
| --- | --- |
| 正式域名健康检查 | 首页、核心页和 `/api/health` 正常 |
| 生产核心路径 | 登录、上传、生成、删除、升级、取消全部通过 |
| sitemap 与 robots | GSC 可读取，无阻断 |
| 日志与告警 | 无持续 5xx、无异常任务堆积 |
| 分析事件 | 首页 CTA、注册、上传、生成、保存、分享、结账均可见 |

## 第 7 天

重点：确认发布稳定、Google 能抓取、用户能完成核心任务。

记录：

- GSC 已发现、已抓取、已索引数量，按语言拆分。
- sitemap 读取状态、canonical、hreflang 和抓取异常。
- 5xx、任务失败、AI 超时、上传失败和 Stripe webhook 失败率。
- 首页到试妆、上传到生成、生成到保存/分享、定价到结账转化率。
- 真实设备 Core Web Vitals，重点关注发现页 LCP。
- 用户支持问题和语言理解问题。

触发条件：

- 任何持续 5xx、支付或额度错误：立即 P0。
- 核心路径失败率明显升高：立即 P1。
- 大量“已抓取但未索引”：检查页面差异、内链和内容价值。

## 第 14 天

重点：确认索引策略和各市场内容表现。

记录：

- 各语言曝光、点击、CTR、平均排名和索引率。
- 高曝光低 CTR 页面及其 title/description。
- 高进入低转化页面及其 CTA、信任说明和加载性能。
- 各语言注册、试妆完成、保存、分享和付费转化。
- 搜索词是否匹配页面真实内容。
- 母语用户反馈和美妆术语问题。

动作：

- 优先修复索引冲突、关键词错配和明显语言问题。
- 对高曝光低 CTR 页面调整标题和描述。
- 为高价值语言市场补充真实案例和内链。

## 第 30 天

重点：确定第一轮增长方向。

形成结论：

- 哪些语言、页面类型和关键词具备继续投入价值。
- 哪些页面应扩写、合并、降权或停止索引。
- 哪个转化环节损失最大。
- 哪些妆容场景、肤色或人群需求最受欢迎。
- 免费、Pro、Premium 的使用和付费结构是否合理。

第一轮增长排期：

- 扩展有效长尾内容集群。
- 增加真实案例、前后对比和本地产品建议。
- 开展 CTA、定价周期和套餐推荐实验。
- 针对发现页继续优化图片加载和真实用户 LCP。
